Cancel Sale Orders

User can cancel multiple sale orders according to specific confirmation/validation condition.


Validation on Confirmation

User can not cancel sale orders without confirmation.


Confirmation for Cancel Sale Orders

User must be apply confirm button for cancel sale orders.


Validation on done Sale Orders

User can not cancel, already has been done sale orders.


Cancelled Sale Orders

User can see the result of cancelled sale orders.
