This module will allow you to add multiple allowances per employee, and to calculate it individually in its payslip



Features:
  1. Multiple Allowances to be configured and added to employee contract.
  2. All allowances are added automatically with wage to the "END OF SERVICE" reward calculation.

Installed with the Module:
  1. A sample allowance "Mobile Allowance".
  2. A Salary Structure and Structure Type.
  3. A Salary rule for "All Allowances", to add the sum of all allowances added to the contract to the payslip.
  4. A Sample salary rule for "Mobile Allowance" to use with any additional allowances.
